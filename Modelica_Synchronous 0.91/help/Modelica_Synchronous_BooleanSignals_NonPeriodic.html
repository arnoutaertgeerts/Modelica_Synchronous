Modelica_Synchronous.BooleanSignals.NonPeriodic.UnitDelay

Delays the clocked input signal for one sample period

Information




This block describes a unit delay:



  // Time domain description
     y(ti) = previous(u(ti))

  // Discrete transfer function
             1
     y(z) = --- * u(z)
             z





that is, the output signal y is the input signal u at the
previous clock tick. At the first clock tick, the output
y is set to parameter y_start.




Extends from Modelica_Synchronous.BooleanSignals.Interfaces.PartialClockedSISO (Block with clocked single input and clocked single output Boolean signals).
